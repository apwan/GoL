The Java Source Code

On this page you can find the source code of the Game of Life.

The org.bitstorm.gameoflife UML Class diagram, created with Enterprise Architect:

The Game of Life is written in Java, version JDK 1.1.

  • org.bitstorm.gameoflife
    • GameOfLife.java - The code for the applet
    • StandaloneGameOfLife.java - The code for the standalone program, extends GameOfLife.java.
    • Cell.java - Represents a cell.
    • CellGrid.java - Represents the visible grid of cells.
    • CellGridCanvas.java - The Canvas showing the grid.
    • GameOfLifeControls.java - The controls (buttons, pull down menu's...).
    • GameOfLifeControlsEvent.java - Event generated by controls.
    • GameOfLifeControlsListener.java - Listener for controls.
    • GameOfLifeGrid.java - Contains the Game of Life algoritm and the current shape.
    • Shape.java - Represents a shape.
    • ShapeCollection.java - Collection of predefined shapes.
    • ShapeException.java - Exceptions of shape operations.
  • org.bitstorm.util
    • AboutDialog.java - About Dialog box class.
    • AlertBox.java - Alert box utility class.
    • EasyFile.java - File reading and writing utility class.
    • ImageComponent.java - Image component utility class.
    • LineEnumerator.java - Enumerates over lines in a text.
    • TextFileDialog.java - Shows text in a dialog.

See also the GameOfLife JavaDoc documentation.

You can also see the older Java 1.0 code.

The software is developed with:

  • IDE: Eclipse
  • Build script: ANT (see the script build.xml)
  • The Windows installer: Nullsoft NSIS (see the script GameOfLife.nsi)
  • JAR Executer: Jarx (Make a JAR file executable)
